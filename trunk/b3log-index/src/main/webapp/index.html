B3log Solo 是款开源（Apache License 2.0）的博客程序，不仅可以运行在 Google App Engine 上，也可以运行在标准 Servlet 容器上。 当前最新版本：0.4.0。
  • Intro
  • About
  • Theme
  • Time Line
  • 基于标签的文章分类
  • Ping Google Blog Search Engine
  • 博客/标签 Atom/RSS 输出
  • Sitemap 输出
  • 评论回复及邮件提醒
  • 自定义页面
  • 置顶/相关/随机/站外相关文章
  • 文章、页面永久链接（Permalink）
  • 文章草稿夹、签名档、更新提示
  • 缓存管理
  • 多用户
  • 多语言
  • 换肤
  • 插件
  • MetaWeblog API
  • 高性能，低配额
